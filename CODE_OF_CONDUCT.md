# Code of Conduct

As contributors and maintainers of this project, we pledge to make participation respectful and harassment-free for everyone.

## Our Standards

Examples of positive behavior:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Giving constructive feedback
- Gracefully accepting criticism

Examples of unacceptable behavior:
- Harassment, insults, or personal attacks
- Publishing othersâ€™ private information
- Disruptive or unprofessional conduct

## Responsibilities

Project maintainers:
- Have the right to remove comments or contributions that violate this Code of Conduct.
- Will enforce this code consistently and fairly.

## Scope

This Code of Conduct applies:
- Within this project and its spaces
- In public spaces when representing the project

## Enforcement

Report unacceptable behavior by opening an Issue or emailing the repository owner.

Thank you for helping keep this community open and respectful.
